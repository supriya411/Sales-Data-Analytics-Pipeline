
📊 Sales Data Analytics Pipeline

🎯 Project Objective
This project demonstrates an end-to-end ETL (Extract, Transform, Load) workflow. I transitioned raw, unorganized sales data into a structured format to identify regional performance trends and core revenue drivers.

 🛠️ Tech Stack
* Language: Python (Pandas)
* Visualization:Matplotlib
* process: Data Cleaning, Outlier Detection, Exploratory Data Analysis (EDA)

 📁 Repository Contents
[Sales_Analysis.ipynb](Sales_Analysis.ipynb):** Full Python logic for data normalization and audit.
[Sales_project.csv](Sales_project.csv):** Validated, clean dataset ready for Power BI/Business Intelligence.

💡 Key Business Insights
* Revenue Drivers: Identified the top-performing categories that account for the majority of profit margins.
* Operational Audit: Cleaned inconsistent date formats and handled missing values to ensure 100% data integrity for reporting.
* Visual Analysis: Built a multi-chart dashboard to compare sales across different timeframes and regions.

📊 Phase 2: Sales Performance Dashboard
Building on the ETL pipeline, this interactive dashboard transforms the processed data into actionable business insights for the 2025-2026 fiscal period.

Key Visualizations & Analytics
Trend Analysis: Area charts showing a major revenue peak in December 2025 ($0.25M).

Market Share: Donut chart highlighting Laptop Pro as the top contributor (64% of total revenue).

Regional Performance: Global map and bar charts identifying Asia-Pacific as the leading market ($0.54M).

Variance Tracking: Waterfall charts illustrating Month-over-Month (MoM) growth across categories.

BI Features
Interactive Slicers: Dynamic filtering by region (APAC, Europe, North America, etc.).

Multi-View Navigation: Dedicated tabs for Sales Performance, Daily Trends, and Product Rankings.

DAX Implementation: Custom measures for YTD Sales, Growth %, and Regional Ranking.
